Session
-------

``kak -s NAME``
  Start new session named NAME
``kak -l``
  List session
``kak -c``
  Connect to session NAME

General
-------

``<c>-o``
  Disable completion for this insert
``<c>-r``
  Paste register followed next
``<a>-;``
  Normal mode for single command

Movement
--------

``<a>-[wbetf]``
  Same as W/B/E/T/F in vim
